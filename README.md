# PHP IMDB GRABBER CLASS
 This php class allows you to grab movie data from imdb.com
 
## Note
this class for movies only
 
## How to Use
```php
<?php
require_once 'imdbGrabber.php';
//Start the class
$imdb = new imdbGrabber();
//Get movie data
$imdb->get('https://www.imdb.com/title/tt0081059/');
//Check if no errors
if (!$imdb->hasError()) {
  echo $imdb->dataJson();
} else {
  echo $imdb->getError();
}
?>
```
